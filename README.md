# Azure Static Website Template
This repository can be used to create a simple Azure hosted static website.

Name: Nicholas Jones

Project Name: Nicholas Jones Dice Rolling

ChatGPT was used for the pleasing aesthetic of the website

Instructions

- index.html is the main webpage
- styles.css is for the visual style
- script.js is for the dice rolling

- Must have a modern web browser (Chrome, Safari, Firefox, etc)

How to Execute

1. Download or clone the project files.
2. Ensure all files are in the same folder.
3. Double-click **index.html**.

The application will open in your web browser.
